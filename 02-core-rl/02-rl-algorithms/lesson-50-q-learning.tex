%% SECTION 1: MOTIVATION AND OVERVIEW
\section{Motivation and Overview}
\label{sec:motivation}

In Lesson 49 we developed SARSA, an on-policy TD control method that
evaluates and improves the policy it actually follows --- including
its exploratory actions. This on-policy nature is a strength when
exploration is costly, but it also means SARSA can only learn from
data generated by the current policy. In many practical scenarios,
however, we want to learn the \emph{optimal} policy from data
generated by a different, more exploratory \emph{behavior policy}.
Consider an autonomous vehicle collecting driving data with a safe,
conservative exploration policy: we would like to use this data to
learn the optimal (and potentially aggressive) driving policy,
without ever executing that aggressive policy during training.

Q-learning resolves this by decoupling the behavior policy
(which generates data) from the target policy (which is being
learned). Its update rule replaces the sampled next action
$A_{t+1}$ in the SARSA target with $\max_{a'} Q(S_{t+1}, a')$ ---
the value of the \emph{best} action, regardless of which action was
actually taken. This seemingly small change has profound consequences:
Q-learning converges to $\Qstar$ no matter what behavior policy
is used (as long as it explores sufficiently), making it the first
and most fundamental off-policy reinforcement learning algorithm.

After establishing Q-learning, we develop Expected SARSA, which
replaces the sampled next action with an expectation over the
policy, and then present the unified view showing that SARSA,
Q-learning, and Expected SARSA are all instances of the same TD
control template with different bootstrap targets. The tools
developed in this lesson will let us prove the convergence of
Q-learning via stochastic approximation and rigorously compare
all three TD control methods.

\begin{rlconnection}[title={Where Q-Learning and Its Variants Appear}]
\begin{itemize}[nosep]
  \item \textbf{Deep Q-Networks (DQN)} combine Q-learning with
    neural network function approximation, experience replay, and
    target networks --- the breakthrough that launched deep RL.
  \item \textbf{Double Q-learning} addresses the maximization bias
    inherent in the $\max$ operator, leading to Double DQN.
  \item \textbf{Expected SARSA} generalizes both SARSA and
    Q-learning and forms the basis of the Rainbow DQN improvements.
  \item \textbf{Off-policy learning} is essential for experience
    replay (reusing past transitions) and learning from
    demonstrations, both foundational techniques in modern deep RL.
\end{itemize}
\end{rlconnection}

\paragraph{Prerequisites.}
Action-value Bellman operators $\Bellman^\policy_Q$ and
$\Bellman^*_Q$ and their $\discount$-contraction properties
(Lesson 49, Lemma 3.3). SARSA update rule and its convergence under

Stochastic approximation: the Robbins--Monro theorem with martingale

  \item \emph{Define} Expected SARSA and \emph{prove} that it has

%% SECTION 3: Q-LEARNING
\section{Q-Learning: Off-Policy TD Control}
\label{sec:qlearning}

\begin{definition}[Q-Learning Update Rule]
\label{def:qlearning}
Given the transition $(S_t, A_t, R_{t+1}, S_{t+1})$, the
\emph{Q-learning update} is
\begin{equation}
  Q(S_t, A_t) \leftarrow Q(S_t, A_t)
  + \alpha_t \bigl[R_{t+1} + \discount \max_{a'} Q(S_{t+1}, a')
  - Q(S_t, A_t)\bigr].
  \label{eq:qlearning}
\end{equation}
The TD error for Q-learning is
$\delta_t = R_{t+1} + \discount \max_{a'} Q(S_{t+1}, a')
- Q(S_t, A_t)$.
\end{definition}

\begin{remark}[Off-Policy Nature]
\label{rem:off_policy}
The critical distinction from SARSA is in the bootstrap target.
SARSA uses $Q(S_{t+1}, A_{t+1})$ where $A_{t+1}$ is drawn from the
\emph{behavior policy}. Q-learning uses $\max_{a'} Q(S_{t+1}, a')$,
which evaluates the \emph{greedy policy} regardless of which action
was actually taken. The behavior policy (which generates transitions)
can be \emph{any} policy with sufficient exploration --- it need not
be the greedy or $\varepsilon$-greedy policy. This decoupling of
behavior and target policies is the hallmark of off-policy learning.
\end{remark}

\begin{remark}[Q-Learning as Stochastic Value Iteration]
\label{rem:value_iteration}
Q-learning can be viewed as a sample-based, asynchronous version of
$Q$-value iteration $Q \leftarrow \Bellman^*_Q Q$. Value iteration
applies $\Bellman^*_Q$ synchronously to all $(s,a)$ pairs using
exact expectations; Q-learning applies it asynchronously to
individually sampled pairs using stochastic estimates.
\end{remark}

\begin{example}[Q-Learning Update]
\label{ex:qlearning_update}
Consider the same $3 \times 1$ grid from Lesson 49 (Example 5.3)
with $\States = \{L, C, R\}$, actions $\{$left, right$\}$,
reward $+1$ at $R$, $\discount = 0.9$, and $Q_0 \equiv 0$.
Suppose at time $t$: $S_t = C$, $A_t = \text{right}$ (exploratory),
$R_{t+1} = 1$, $S_{t+1} = R$. With $\alpha = 0.1$:
\begin{align*}
  Q_1(C, \text{right})
  &= 0 + 0.1[1 + 0.9 \cdot \max(Q_0(R, \text{left}),
     Q_0(R, \text{right})) - 0] \\
  &= 0.1[1 + 0.9 \cdot 0] = 0.1.
\end{align*}
This is the same as the SARSA update in this case because
$Q_0 \equiv 0$. The difference emerges later, when
$Q(R, \cdot) \neq 0$: Q-learning uses
$\max_{a'} Q(R, a')$ while SARSA uses the randomly sampled
$Q(R, A_{t+1})$.
\end{example}

We now prove the central convergence result of this lesson.

%% SECTION 4: CONVERGENCE OF Q-LEARNING
\section{Convergence of Q-Learning}
\label{sec:convergence}

\begin{assumption}[Q-Learning Convergence Conditions]
\label{ass:qlearning}
\begin{enumerate}[nosep]
  \item The MDP has finite state space $\States$ and finite action
    space $\Actions$.
  \item Rewards are bounded: $|r(s,a,s')| \leq R_{\max}$ for all
    $(s,a,s')$.
  \item Every state--action pair $(s,a)$ is visited infinitely often.
  \item Step sizes satisfy $\sum_n \alpha_n(s,a) = \infty$ and
    $\sum_n \alpha_n^2(s,a) < \infty$ for each
    $(s,a) \in \States \times \Actions$.
\end{enumerate}
\end{assumption}

\begin{keyresult}[title={Convergence of Q-Learning (Watkins \& Dayan 1992)}]
\begin{theorem}[Convergence of Q-Learning]
\label{thm:qlearning}
Under Assumption~\ref{ass:qlearning}, the Q-learning iterates
converge to the optimal action-value function almost surely:
\begin{equation}
  Q_t(s,a) \cas \Qstar(s,a)
  \quad \text{for all } (s,a) \in \States \times \Actions.
  \label{eq:qlearning_convergence}
\end{equation}
\end{theorem}
\end{keyresult}

\begin{proof}
We follow \citet{watkins1992} and \citet{tsitsiklis1994}, casting
Q-learning as a stochastic approximation and verifying all conditions
of the Robbins--Monro theorem (Lesson 31, Theorem 3.7).

\textbf{Step 1: Express as stochastic approximation.}
At the $k$-th visit to state--action pair $(s,a)$ (occurring at
global time $t_k$), the Q-learning update reads:
\begin{equation}
  Q_{k+1}(s,a) = Q_k(s,a)
  + \alpha_k(s,a)\bigl[(\Bellman^*_Q Q_k)(s,a) - Q_k(s,a)
  + w_k(s,a)\bigr],
  \label{eq:ql_sa}
\end{equation}
where the noise is
\begin{equation}
  w_k(s,a) = R_{t_k+1} + \discount \max_{a'} Q_k(S_{t_k+1}, a')
  - (\Bellman^*_Q Q_k)(s,a).
  \label{eq:ql_noise}
\end{equation}
The Bellman optimality operator on $Q$-values is
$(\Bellman^*_Q Q)(s,a) = \sum_{s'} \Trans(s' \mid s,a)
[r(s,a,s') + \discount \max_{a'} Q(s',a')]$
(Lesson 49, Definition 3.2).

\textbf{Step 2: Verify the contraction property.}
By Lesson 49, Lemma 3.3, $\Bellman^*_Q$ is a
$\discount$-contraction in $\|\cdot\|_\infty$ with unique fixed
point $\Qstar$. The driving function
$h(Q)(s,a) = (\Bellman^*_Q Q)(s,a) - Q(s,a)$ has the unique root
$Q = \Qstar$.

For stability, we verify the contraction condition for the expected
update. For any $\alpha \in (0,1]$:
\begin{align}
  \|Q + \alpha\,h(Q) - \Qstar\|_\infty
  &= \|(1-\alpha)Q + \alpha\,\Bellman^*_Q Q - \Qstar\|_\infty
    \nonumber\\
  &= \|(1-\alpha)(Q - \Qstar) + \alpha(\Bellman^*_Q Q
    - \Bellman^*_Q \Qstar)\|_\infty \nonumber\\
  &\leq (1-\alpha)\|Q - \Qstar\|_\infty
    + \alpha\discount\|Q - \Qstar\|_\infty \nonumber\\
  &= (1 - \alpha(1-\discount))\|Q - \Qstar\|_\infty.
  \label{eq:ql_stability}
\end{align}
Since $\alpha(1-\discount) > 0$, the expected update contracts the
error, confirming that $h$ drives iterates toward $\Qstar$.

\textbf{Step 3: Verify martingale noise.}
Let $\mathcal{F}_k$ denote the filtration containing all information
up to the $k$-th visit to $(s,a)$ (including $Q_k$ and
$S_{t_k} = s, A_{t_k} = a$). Then:
\begin{align}
  \E[w_k(s,a) \mid \mathcal{F}_k]
  &= \E\bigl[R_{t_k+1} + \discount \max_{a'} Q_k(S_{t_k+1}, a')
     \mid S_{t_k} = s, A_{t_k} = a, Q_k\bigr]
     - (\Bellman^*_Q Q_k)(s,a) \nonumber\\
  &= \sum_{s'} \Trans(s' \mid s,a)
     \bigl[r(s,a,s') + \discount \max_{a'} Q_k(s',a')\bigr]
     - (\Bellman^*_Q Q_k)(s,a) \nonumber\\
  &= (\Bellman^*_Q Q_k)(s,a) - (\Bellman^*_Q Q_k)(s,a)
   = 0.
  \label{eq:ql_noise_mg}
\end{align}
Thus $\{w_k(s,a)\}$ is a martingale difference sequence with respect
to $\{\mathcal{F}_k\}$.

\textbf{Step 4: Bound the noise variance.}
Since $|R_{t+1}| \leq R_{\max}$ and the $Q$-iterates remain bounded
by $R_{\max}/(1-\discount)$ (established by induction using the
contraction~\eqref{eq:ql_stability}):
\begin{equation}
  |w_k(s,a)| \leq R_{\max} + \frac{\discount R_{\max}}{1-\discount}
  + \frac{R_{\max}}{1-\discount}
  = \frac{(2-\discount)R_{\max}}{1-\discount}.
  \label{eq:ql_noise_bound}
\end{equation}
Therefore $\E[w_k^2(s,a) \mid \mathcal{F}_k] \leq C$ for a
constant $C = [(2-\discount)R_{\max}/(1-\discount)]^2$.

\textbf{Step 5: Apply the Robbins--Monro theorem.}
All five conditions of the Robbins--Monro theorem are satisfied:
\begin{enumerate}[nosep]
  \item Step sizes: $\sum_k \alpha_k(s,a) = \infty$ and
    $\sum_k \alpha_k^2(s,a) < \infty$ by assumption.
  \item Martingale noise: $\E[w_k \mid \mathcal{F}_k] = 0$
    (Step 3).
  \item Unique root: $h(Q) = 0$ if and only if $Q = \Qstar$
    (Step 2).
  \item Stability: the contraction~\eqref{eq:ql_stability}
    ensures the expected update contracts toward $\Qstar$.
  \item Bounded noise: $\E[w_k^2 \mid \mathcal{F}_k] \leq C$
    (Step 4).
\end{enumerate}
Therefore $Q_k(s,a) \to \Qstar(s,a)$ almost surely for all
$(s,a) \in \States \times \Actions$.
\end{proof}

\begin{remark}[No GLIE Needed]
\label{rem:no_glie}
Unlike SARSA, Q-learning does not require the GLIE condition.
The behavior policy need not converge to greedy --- it only needs
to visit every state--action pair infinitely often
(Assumption~\ref{ass:qlearning}, Condition 3). This is a weaker
requirement than GLIE's Condition 2
($\varepsilon_t \to 0$). In SARSA, the policy must converge to
greedy so that the exploration perturbation vanishes
(Lesson 49, proof of Theorem 6.1, Step 5); in Q-learning, the
$\max$ operator directly targets $\Qstar$ without any
perturbation.
\end{remark}

\begin{example}[Off-Policy Data Collection]
\label{ex:off_policy}
Suppose an agent explores with a uniformly random policy
$\policy_b(a \mid s) = 1/|\Actions|$ for all $(s,a)$. Q-learning
applied to the resulting transitions still converges to $\Qstar$,
because Assumption~\ref{ass:qlearning} is satisfied: the uniform
policy visits all state--action pairs infinitely often. SARSA, by
contrast, would converge to $Q^{\policy_b}$ (the value of the
random policy), which is far from optimal.
\end{example}

\begin{rlconnection}[title={From Q-Learning to DQN}]
Q-learning's convergence guarantee holds for the tabular case.
When combined with neural network function approximation
(DQN, \citet[Chapter 16]{suttonbarto2018}), two additional
techniques are needed to maintain stability:
\begin{itemize}[nosep]
  \item \textbf{Experience replay}: stores transitions in a buffer
    and samples mini-batches for updates, breaking temporal
    correlation and enabling off-policy reuse.
  \item \textbf{Target networks}: uses a slowly updated copy of the
    $Q$-network for the bootstrap target
    $\max_{a'} Q_{\text{target}}(s', a')$, reducing the moving-target
    problem.
\end{itemize}
Both techniques exploit Q-learning's off-policy nature: experience
replay reuses past transitions (generated by earlier policies), and
the target network evaluates a frozen (hence different) policy.
\end{rlconnection}

%% SECTION 5: MAXIMIZATION BIAS
\section{Maximization Bias and Double Q-Learning}
\label{sec:max_bias}

The $\max$ operator in Q-learning introduces a systematic positive
bias when the $Q$-values are noisy estimates.

\begin{proposition}[Maximization Bias]
\label{prop:max_bias}
Let $X_1, \ldots, X_n$ be random variables with
$\E[X_i] = q_i$. Then
\begin{equation}
  \E\!\left[\max_{i} X_i\right] \geq \max_{i} \E[X_i]
  = \max_i q_i.
  \label{eq:max_bias}
\end{equation}
Equality holds if and only if $\max_i X_i = X_{i^*}$ almost surely
for the same index $i^*$ (i.e., the maximizer is deterministic) or
all $X_i$ are equal almost surely.
\end{proposition}

\begin{proof}
For any fixed index $j$: $\max_i X_i \geq X_j$. Taking
expectations: $\E[\max_i X_i] \geq \E[X_j] = q_j$. Since this
holds for all $j$, $\E[\max_i X_i] \geq \max_j q_j$.

For the equality condition, define $Y = \max_i X_i - X_{i^*}$ where
$i^* = \argmax_i q_i$. Then $Y \geq 0$ and
$\E[Y] = \E[\max_i X_i] - q_{i^*}$. If $\max_i X_i = X_{i^*}$
a.s., then $Y = 0$ a.s.\ and equality holds. Otherwise,
$\Prob(Y > 0) > 0$, implying $\E[Y] > 0$ and strict inequality.
\end{proof}

\begin{example}[Maximization Bias in a Simple MDP]
\label{ex:max_bias}
Consider a state $s$ with two actions, where the true values are
$\Qstar(s,1) = \Qstar(s,2) = 0$, but the current noisy estimates
are $Q(s,1) = 0.3$ and $Q(s,2) = -0.2$. Then
$\max_a Q(s,a) = 0.3$, which overestimates the true value
$\max_a \Qstar(s,a) = 0$. Q-learning uses this inflated estimate
in its bootstrap target, potentially propagating and amplifying the
overestimation.
\end{example}

\begin{definition}[Double Q-Learning]
\label{def:double_q}
\emph{Double Q-learning} maintains two independent $Q$-value
estimates $Q^A$ and $Q^B$. At each step, with equal probability,
one estimate is used to select the action and the other to evaluate
it:
\begin{equation}
  Q^A(S_t, A_t) \leftarrow Q^A(S_t, A_t)
  + \alpha\bigl[R_{t+1} + \discount\, Q^B\bigl(S_{t+1},
    \argmax_{a'} Q^A(S_{t+1}, a')\bigr) - Q^A(S_t, A_t)\bigr],
  \label{eq:double_q}
\end{equation}
with a symmetric update for $Q^B$ chosen with probability $1/2$.
\end{definition}

\begin{proposition}[Double Q-Learning Removes Maximization Bias]
\label{prop:double_unbiased}
In Double Q-learning, the expected bootstrap target is
\begin{equation}
  \E\bigl[Q^B\bigl(S_{t+1}, \argmax_{a'} Q^A(S_{t+1}, a')\bigr)
    \mid S_{t+1}\bigr],
  \label{eq:double_target}
\end{equation}
which is an unbiased estimate of the value of the action selected
by $Q^A$, rather than an overestimate of $\max_{a'} \Qstar(S_{t+1},
a')$.
\end{proposition}

\begin{proof}
Let $a^* = \argmax_{a'} Q^A(S_{t+1}, a')$. Since $Q^A$ and $Q^B$
are updated on independent samples (by the random assignment
mechanism), $Q^B(S_{t+1}, a^*)$ is an unbiased estimate of
the value of action $a^*$ (not of $\max_{a'} \Qstar(s,a')$).
Specifically:
\begin{equation}
  \E[Q^B(S_{t+1}, a^*) \mid S_{t+1}, Q^A]
  = \Qstar(S_{t+1}, a^*) + \text{estimation error of } Q^B,
\end{equation}
where the estimation error has zero mean under appropriate
conditions. The key point is that $Q^B$ does not select $a^*$ ---
it merely evaluates it --- so there is no selection-evaluation
coupling that causes maximization bias.
\end{proof}

\begin{warning}[title={Q-Learning Can Significantly Overestimate Values}]
In MDPs with many actions and noisy rewards, maximization bias
causes Q-learning to systematically overestimate $\Qstar$,
sometimes by large margins. This can lead to poor policy
performance: the agent believes certain actions are much better
than they actually are and repeatedly selects them. Double
Q-learning (or its deep RL variant, Double DQN) is the standard
remedy.
\end{warning}

%% SECTION 6: EXPECTED SARSA
\section{Expected SARSA}
\label{sec:expected_sarsa}

Expected SARSA replaces the sampled next action in the SARSA target
with an expectation over the policy, reducing variance while
maintaining a flexible on/off-policy character.

\begin{definition}[Expected SARSA Update Rule]
\label{def:expected_sarsa}
The \emph{Expected SARSA update} is
\begin{equation}
  Q(S_t, A_t) \leftarrow Q(S_t, A_t)
  + \alpha_t\!\left[R_{t+1}
  + \discount \sum_{a'} \policy(a' \mid S_{t+1}) Q(S_{t+1}, a')
  - Q(S_t, A_t)\right].
  \label{eq:expected_sarsa}
\end{equation}
\end{definition}

\begin{remark}[Expected SARSA Subsumes Q-Learning]
\label{rem:esarsa_qlearning}
When $\policy$ is the greedy policy with respect to $Q$, we have
$\sum_{a'} \policy(a' \mid s') Q(s',a') = \max_{a'} Q(s',a')$,
and Expected SARSA reduces to Q-learning. When $\policy$ is the
behavior policy (e.g., $\varepsilon$-greedy), Expected SARSA
is on-policy and evaluates the actual behavior.
\end{remark}

\subsection{Variance Reduction}
\label{subsec:esarsa_variance}

\begin{keyresult}[title={Expected SARSA Has Lower Variance Than SARSA}]
\begin{proposition}[Variance Reduction]
\label{prop:esarsa_variance}
For any policy $\policy$ and any action-value function $Q$:
\begin{equation}
  \Var\!\left(R_{t+1} + \discount \sum_{a'} \policy(a' \mid S_{t+1})
    Q(S_{t+1}, a') \;\middle|\; S_t, A_t\right)
  \leq \Var\!\left(R_{t+1} + \discount Q(S_{t+1}, A_{t+1})
    \;\middle|\; S_t, A_t\right).
  \label{eq:esarsa_variance}
\end{equation}
\end{proposition}
\end{keyresult}

\begin{proof}
By the law of total variance, conditioning on $S_{t+1}$:
\begin{align}
  &\Var\!\left(R_{t+1} + \discount Q(S_{t+1}, A_{t+1})
    \;\middle|\; S_t, A_t\right) \nonumber\\
  &= \E\!\left[\Var\!\left(\discount Q(S_{t+1}, A_{t+1})
    \;\middle|\; S_{t+1}\right) \;\middle|\; S_t, A_t\right]
  + \Var\!\left(R_{t+1} + \discount \E[Q(S_{t+1}, A_{t+1})
    \mid S_{t+1}] \;\middle|\; S_t, A_t\right).
  \label{eq:esarsa_var_step1}
\end{align}
Since $A_{t+1} \sim \policy(\cdot \mid S_{t+1})$, the conditional
expectation is
$\E[Q(S_{t+1}, A_{t+1}) \mid S_{t+1}]
= \sum_{a'} \policy(a' \mid S_{t+1}) Q(S_{t+1}, a')$.
Therefore:
\begin{align}
  &= \underbrace{\discount^2
    \E\!\left[\Var_{A_{t+1} \sim \policy}\!\left(Q(S_{t+1}, A_{t+1})
    \;\middle|\; S_{t+1}\right) \;\middle|\; S_t, A_t\right]
  }_{\geq\, 0}
  + \Var\!\left(R_{t+1} + \discount \sum_{a'} \policy(a' \mid
    S_{t+1}) Q(S_{t+1}, a') \;\middle|\; S_t, A_t\right).
  \label{eq:esarsa_var_step2}
\end{align}
The first term is the expected variance of $Q$ over the action
choice $A_{t+1}$, which is non-negative. The second term is exactly
the variance of the Expected SARSA target. Thus the SARSA variance
exceeds the Expected SARSA variance by the non-negative first term.
\end{proof}

\begin{example}[Variance Comparison]
\label{ex:esarsa_variance}
Consider a state $s'$ with $|\Actions| = 4$ and $Q$-values
$(10, 10, 0, 0)$ under an $\varepsilon$-greedy policy with
$\varepsilon = 0.2$ (so $\policy(a_1 \mid s') = 0.85$,
$\policy(a_i \mid s') = 0.05$ for $i = 2,3,4$).

SARSA samples $A_{t+1} \sim \policy$, yielding $Q(s', A_{t+1})$
that is either 10 (with probability 0.90) or 0 (with probability
0.10). The conditional variance of $Q(s', A_{t+1})$ is
$0.9 \cdot 10^2 + 0.1 \cdot 0^2 - (0.9 \cdot 10)^2 = 90 - 81 = 9$.

Expected SARSA deterministically computes
$\sum_{a'} \policy(a' \mid s') Q(s', a')
= 0.85(10) + 0.05(10) + 0.05(0) + 0.05(0) = 9.0$.
This contributes zero variance from the action selection.
\end{example}

\subsection{Convergence of Expected SARSA}
\label{subsec:esarsa_convergence}

\begin{theorem}[Convergence of Expected SARSA]
\label{thm:esarsa}
Under the same conditions as SARSA (Lesson 49,
Assumption~\ref{ass:sarsa} with $\policy_t$ satisfying GLIE),
Expected SARSA converges to $\Qstar$ almost surely. If the policy
$\policy$ is fixed, Expected SARSA converges to $\Qpi$.
\end{theorem}

\begin{proof}
The Expected SARSA update targets the Bellman expectation operator
$\Bellman^{\policy}_Q$ (Lesson 49, Definition 3.1):
\begin{equation}
  (\Bellman^{\policy}_Q Q)(s,a)
  = \sum_{s'} \Trans(s' \mid s,a) \left[
    r(s,a,s') + \discount \sum_{a'} \policy(a' \mid s') Q(s',a')
  \right].
  \label{eq:esarsa_operator}
\end{equation}
This is a $\discount$-contraction with fixed point $\Qpi$
(Lesson 49, Lemma 3.3).

The update noise is
$w_t = R_{t+1} + \discount \sum_{a'} \policy(a' \mid S_{t+1})
Q(S_{t+1}, a') - (\Bellman^{\policy}_Q Q)(S_t, A_t)$. This
satisfies $\E[w_t \mid S_t, A_t, Q] = 0$ because the randomness
comes only from the transition $S_{t+1}$ (the action sum is
deterministic given $S_{t+1}$), and the expectation over
$S_{t+1}$ yields $(\Bellman^{\policy}_Q Q)(S_t, A_t)$ by
definition.

For fixed $\policy$, the Robbins--Monro theorem gives
$Q_t \to \Qpi$ a.s. Under GLIE ($\policy_t \to \policy^*_{\text{greedy}}$),
Expected SARSA targets converge to
$\max_{a'} Q(S_{t+1}, a')$, and the same perturbed stochastic
approximation argument as in SARSA's convergence proof
(Lesson 49, Theorem 6.1) yields convergence to $\Qstar$.
\end{proof}

\begin{rlconnection}[title={Expected SARSA in Practice}]
Expected SARSA often outperforms both SARSA and Q-learning
empirically:
\begin{itemize}[nosep]
  \item Compared to SARSA: lower variance from eliminating the
    action sampling noise (Proposition~\ref{prop:esarsa_variance}).
  \item Compared to Q-learning: no maximization bias when $\policy$
    is not the greedy policy.
  \item The additional computational cost is $O(|\Actions|)$ per
    update (summing over actions), which is negligible for small
    action spaces.
\end{itemize}
\end{rlconnection}

%% SECTION 7: UNIFIED VIEW OF TD CONTROL
\section{Unified View of TD Control}
\label{sec:unified}

All three TD control algorithms share a common structure that
differs only in the bootstrap target.

\begin{keyresult}[title={Unified TD Control Framework}]
\begin{theorem}[Unified TD Control Update]
\label{thm:unified}
SARSA, Q-learning, and Expected SARSA can all be written as
\begin{equation}
  Q(S_t, A_t) \leftarrow Q(S_t, A_t)
  + \alpha_t\bigl[R_{t+1} + \discount\,\mathrm{Target}(S_{t+1})
  - Q(S_t, A_t)\bigr],
  \label{eq:unified}
\end{equation}
where the target function varies across algorithms:
\begin{center}
\begin{tabular}{lccc}
\toprule
\textbf{Algorithm} & $\mathrm{Target}(S_{t+1})$
  & \textbf{On/Off-Policy} & \textbf{Converges to} \\
\midrule
SARSA
  & $Q(S_{t+1}, A_{t+1})$, $A_{t+1} \sim \policy$
  & On-policy
  & $\Qstar$ (GLIE) \\
Q-Learning
  & $\max_{a'} Q(S_{t+1}, a')$
  & Off-policy
  & $\Qstar$ \\
Expected SARSA
  & $\sum_{a'} \policy(a' \mid S_{t+1}) Q(S_{t+1}, a')$
  & Either
  & $\Qpi$ or $\Qstar$ \\
\bottomrule
\end{tabular}
\end{center}
\end{theorem}
\end{keyresult}

\begin{proof}
The unified form follows by inspection of Definitions
5.1 (Lesson 49), \ref{def:qlearning}, and
\ref{def:expected_sarsa}. The convergence column follows from
Lesson 49, Theorem 6.1, Theorem~\ref{thm:qlearning}, and
Theorem~\ref{thm:esarsa} respectively.
\end{proof}

\begin{proposition}[Relationships Between the Targets]
\label{prop:target_relations}
The three targets satisfy the following relationships:
\begin{enumerate}
  \item $\E_\policy[Q(S_{t+1}, A_{t+1}) \mid S_{t+1}]
    = \sum_{a'} \policy(a' \mid S_{t+1}) Q(S_{t+1}, a')$.
    That is, the Expected SARSA target is the conditional
    expectation of the SARSA target.
  \item If $\policy$ is the greedy policy with respect to $Q$:
    $\sum_{a'} \policy(a' \mid s') Q(s',a')
    = \max_{a'} Q(s', a')$. That is, Expected SARSA reduces
    to Q-learning.
  \item If $\policy = \policy_b$ (the behavior policy):
    Expected SARSA is on-policy, evaluating $\policy_b$.
  \item The SARSA target is an unbiased but noisy estimate of the
    Expected SARSA target:
    $\Var(Q(s', A_{t+1}) \mid s') \geq 0$ with equality
    iff $Q(s', \cdot)$ is constant or $\policy$ is deterministic.
\end{enumerate}
\end{proposition}

\begin{proof}
\begin{enumerate}
  \item Direct computation:
    $\E_\policy[Q(S_{t+1}, A_{t+1}) \mid S_{t+1} = s']
    = \sum_{a'} \policy(a' \mid s') Q(s', a')$.
  \item If $\policy(a \mid s') = \indicator\{a = \argmax_{a'} Q(s',
    a')\}$, then $\sum_{a'} \policy(a' \mid s') Q(s', a')
    = Q(s', \argmax_{a'} Q(s', a')) = \max_{a'} Q(s', a')$.
  \item By definition, on-policy means the target policy equals
    the behavior policy. If $\policy = \policy_b$, Expected SARSA's
    target evaluates $\policy_b$.
  \item From (1), $Q(s', A_{t+1})$ is a sample from a distribution
    with mean $\sum_{a'} \policy(a' \mid s') Q(s', a')$. Its
    variance is zero iff $Q(s', a)$ takes the same value for all
    $a$ with $\policy(a \mid s') > 0$, or $\policy$ concentrates
    on a single action. \qedhere
\end{enumerate}
\end{proof}

\begin{example}[Unified View on a Two-Action MDP]
\label{ex:unified}
Consider $S_{t+1} = s'$ with $Q(s', a_1) = 5$,
$Q(s', a_2) = 3$, and $\varepsilon$-greedy policy with
$\varepsilon = 0.1$. Then $\policy(a_1 \mid s') = 0.95$,
$\policy(a_2 \mid s') = 0.05$.

\begin{center}
\begin{tabular}{lcc}
\toprule
\textbf{Algorithm} & \textbf{Target} & \textbf{Value} \\
\midrule
Q-learning & $\max(5, 3)$ & $5.00$ \\
Expected SARSA & $0.95(5) + 0.05(3)$ & $4.90$ \\
SARSA (if $A_{t+1} = a_1$) & $Q(s', a_1)$ & $5.00$ \\
SARSA (if $A_{t+1} = a_2$) & $Q(s', a_2)$ & $3.00$ \\
\midrule
SARSA (expected) & $\E[Q(s', A_{t+1})]$ & $4.90$ \\
\bottomrule
\end{tabular}
\end{center}
The Expected SARSA target (4.90) lies between Q-learning (5.00)
and the average SARSA target, but with zero action-sampling
variance. Q-learning's target (5.00) exceeds Expected SARSA's
(4.90), reflecting the greedy maximization.
\end{example}

\begin{remark}[The Continuum of TD Control]
\label{rem:continuum}
The three algorithms form a spectrum parameterized by the ``degree
of greediness'' in the bootstrap target:
\begin{itemize}[nosep]
  \item \textbf{SARSA}: samples from the behavior policy
    (most stochastic, on-policy).
  \item \textbf{Expected SARSA}: averages over the behavior policy
    (deterministic given $S_{t+1}$, flexible on/off-policy).
  \item \textbf{Q-learning}: maximizes over actions
    (most greedy, off-policy).
\end{itemize}
Expected SARSA smoothly interpolates: as $\varepsilon \to 0$ in the
$\varepsilon$-greedy policy, Expected SARSA approaches Q-learning;
for any fixed $\varepsilon > 0$, it evaluates the exploratory policy.
\end{remark}

%% SECTION 8: CONVERGENCE COMPARISON
\section{Convergence Comparison}
\label{sec:comparison}

\begin{proposition}[Summary of Convergence Properties]
\label{prop:comparison}
The convergence properties of the three TD control algorithms differ
along three axes:
\begin{center}
\begin{tabular}{lccc}
\toprule
& \textbf{SARSA} & \textbf{Q-Learning} & \textbf{Expected SARSA} \\
\midrule
Converges to & $\Qstar$ (GLIE) & $\Qstar$ & $\Qpi$ or $\Qstar$ \\
Needs GLIE? & Yes & No & Depends on $\policy$ \\
Bellman operator & $\Bellman^{\policy_t}_Q$ & $\Bellman^*_Q$
  & $\Bellman^\policy_Q$ \\
Update variance & High & Medium & Low \\
Max.\ bias & No & Yes & No (if $\policy \neq$ greedy) \\
\bottomrule
\end{tabular}
\end{center}
\end{proposition}

\begin{proof}
The convergence targets and GLIE requirements follow from
Theorems 6.1 (Lesson 49), \ref{thm:qlearning},
and~\ref{thm:esarsa}. The Bellman operators follow from the
definitions. The variance ordering
$\Var(\text{Expected SARSA}) \leq \Var(\text{SARSA})$ is
Proposition~\ref{prop:esarsa_variance}. Q-learning's variance is
intermediate because it eliminates the action-selection variance
of SARSA (by using $\max$) but introduces variance from the
$\max$ operator itself when the estimates are noisy. The
maximization bias column follows from
Proposition~\ref{prop:max_bias}: the $\max$ in Q-learning causes
positive bias, while the expectation in Expected SARSA and the
sampling in SARSA do not.
\end{proof}

\begin{example}[Algorithm Comparison on a Cliff-Walking Task]
\label{ex:cliff}
Consider a $4 \times 12$ gridworld with a cliff along the bottom
edge (positions $(1,1)$ through $(1,10)$). The start is $(1,0)$
and the goal is $(1,11)$. Stepping onto the cliff gives $-100$
reward and returns the agent to start; all other steps give $-1$.
With $\varepsilon = 0.1$ and $\discount = 1$:
\begin{itemize}[nosep]
  \item \textbf{Q-learning} learns the optimal path along the
    cliff edge (shortest), but the agent falls off frequently
    during training due to $\varepsilon$-greedy exploration.
  \item \textbf{SARSA} learns a safer path one row above the
    cliff, accounting for the 10\% chance of random steps.
    Online performance during training is better.
  \item \textbf{Expected SARSA} behaves similarly to SARSA
    but with less variance in the updates, often achieving
    the best online performance.
\end{itemize}
After $\varepsilon \to 0$, all three converge to the same
optimal policy.
\end{example}

%% SECTION 9: EXERCISES
\section{Exercises}
\label{sec:exercises}

\begin{exercise}[Q-Learning TD Error as Martingale Difference]
\textnormal{(\(*\))} \\
Let $\delta_t = R_{t+1} + \discount \max_{a'} Q(S_{t+1}, a')
- Q(S_t, A_t)$ be the Q-learning TD error, and suppose
$Q = \Qstar$. Show that $\E[\delta_t \mid S_t, A_t] = 0$, i.e.,
the TD error under the optimal $Q$-function is a martingale
difference sequence.
\end{exercise}

\begin{exercise}[Expected SARSA on a Simple MDP]
\textnormal{(\(*\))} \\
Consider $\States = \{A, B\}$, $\Actions = \{1, 2\}$, transitions:
both actions in $A$ lead to $B$; action 1 in $B$ returns to $A$,
action 2 in $B$ stays in $B$. Rewards: $r(A, 1) = 1$,
$r(A, 2) = 0$, $r(B, 1) = 0$, $r(B, 2) = 2$. Let
$\discount = 0.9$, $Q_0 \equiv 0$, and $\policy$ be the
$\varepsilon$-greedy policy with $\varepsilon = 0.2$.
Compute the first Expected SARSA update for each state--action pair
visited in the trajectory $(A, 1, 1, B, 2, 2, B)$.
\end{exercise}

\begin{exercise}[Maximization Bias Quantification]
\textnormal{(\(**\))} \\
Let $Q_1, \ldots, Q_n$ be independent estimates of the same true
value $q$, each with $\E[Q_i] = q$ and $\Var(Q_i) = \sigma^2$.
\begin{enumerate}[nosep]
  \item[(a)] Show that $\E[\max_i Q_i] \geq q$ with equality
    iff $\sigma = 0$ or $n = 1$.
  \item[(b)] For the special case $n = 2$ and
    $Q_i \sim \mathcal{N}(q, \sigma^2)$, show that
    $\E[\max(Q_1, Q_2)] = q + \sigma/\sqrt{\pi}$.
  \item[(c)] Conclude that the maximization bias grows with
    $\sigma$ and $n$.
\end{enumerate}
\textit{Hint for (b):} Use the identity
$\E[\max(X,Y)] = \E[X] + \E[(Y-X)^+]$ and the fact that
$Y - X \sim \mathcal{N}(0, 2\sigma^2)$.
\end{exercise}

\begin{exercise}[Double Q-Learning Convergence]
\textnormal{(\(**\))} \\
Show that Double Q-learning (Definition~\ref{def:double_q})
converges to $\Qstar$ under the same conditions as standard
Q-learning (Assumption~\ref{ass:qlearning}). Specifically:
\begin{enumerate}[nosep]
  \item[(a)] Write the Double Q-learning update for $Q^A$ in the
    stochastic approximation form $Q^A_{k+1} = Q^A_k + \alpha_k
    [h(Q^A_k, Q^B_k) + w_k]$ and identify $h$ and $w_k$.
  \item[(b)] Show that $h(Q, Q) = (\Bellman^*_Q Q)(s,a)
    - Q(s,a)$ when both estimates agree.
  \item[(c)] Argue that $Q^A$ and $Q^B$ converge to the same
    limit $\Qstar$ by a coupled stochastic approximation argument.
\end{enumerate}
\end{exercise}

\begin{exercise}[Variance Ordering Proof Extension]
\textnormal{(\(***\))} \\
Extend the variance comparison of
Proposition~\ref{prop:esarsa_variance} to the $n$-step case.
Define $n$-step SARSA and $n$-step Expected SARSA targets, and
prove that the Expected SARSA variant has lower variance for any
$n \geq 1$.

\textit{Hint:} Apply the law of total variance iteratively,
conditioning on $(S_{t+1}, \ldots, S_{t+n})$ and then on the
actions.
\end{exercise}

\begin{exercise}[Constructing a Counterexample for Off-Policy Divergence]
\textnormal{(\(***\))} \\
Consider a two-state MDP with states $\{s_1, s_2\}$ and two
actions. Construct specific transition probabilities and rewards
such that Q-learning with a uniform random behavior policy converges
in fewer iterations than SARSA with $\varepsilon_t = 1/t$
($\varepsilon$-greedy). Conversely, construct a different MDP where
SARSA with $\varepsilon_t = 1/t$ achieves lower cumulative regret
during learning than Q-learning (by exploiting the on-policy safety
advantage).

\textit{Hint:} For the second part, make exploratory actions in
certain states very costly.
\end{exercise}

%% SECTION 10: SUMMARY
\section{Summary}
\label{sec:summary}

\begin{itemize}
  \item \textbf{Q-learning} updates action values using the target
    $R_{t+1} + \discount \max_{a'} Q(S_{t+1}, a')$, directly
    targeting the Bellman optimality operator $\Bellman^*_Q$
    regardless of the behavior policy. It is the canonical
    off-policy TD control algorithm.

  \item \textbf{Q-learning converges to $\Qstar$} almost surely
    under finite MDP, bounded rewards, infinite visitation of all
    state--action pairs, and Robbins--Monro step sizes
    (Theorem~\ref{thm:qlearning}). No GLIE condition is needed ---
    the behavior policy can be any sufficiently exploratory policy.

  \item \textbf{Maximization bias}: the $\max$ operator in
    Q-learning causes systematic overestimation of $\Qstar$ when
    $Q$-estimates are noisy
    (Proposition~\ref{prop:max_bias}).
    \textbf{Double Q-learning} corrects this by decoupling action
    selection from action evaluation using two independent estimates.

  \item \textbf{Expected SARSA} uses the target
    $\sum_{a'} \policy(a' \mid S_{t+1}) Q(S_{t+1}, a')$,
    which is the conditional expectation of the SARSA target given
    $S_{t+1}$. It has \emph{lower variance} than SARSA
    (Proposition~\ref{prop:esarsa_variance}) and \emph{no
    maximization bias} (when $\policy$ is not fully greedy).

  \item The \textbf{unified TD control framework}
    (Theorem~\ref{thm:unified}) shows that SARSA, Q-learning, and
    Expected SARSA share the same update structure
    $Q(S_t,A_t) \leftarrow Q(S_t,A_t) + \alpha[\delta_t]$, differing
    only in the bootstrap target at $S_{t+1}$.

  \item Expected SARSA \textbf{interpolates} between SARSA and
    Q-learning: it reduces to Q-learning when the target policy is
    greedy, and to on-policy evaluation when the target policy
    equals the behavior policy.

  \item All three algorithms operate on the same Bellman operator
    structure ($\Bellman^\policy_Q$ or $\Bellman^*_Q$), which
    are $\discount$-contractions, providing the stability condition
    for convergence via stochastic approximation.
\end{itemize}

%% SECTION 11: FURTHER READING
\section{Further Reading}
\label{sec:further_reading}

\begin{itemize}
  \item \textbf{Lessons 51--53} extend TD control to the function
    approximation setting, where the deadly triad of function
    approximation, bootstrapping, and off-policy learning can cause
    divergence. Q-learning is particularly susceptible because it is
    both bootstrapped and off-policy.

  \item \citet{watkins1992} is the original Q-learning paper,
    introducing the algorithm and proving convergence for the
    tabular case.

  \item \citet{tsitsiklis1994} provides the most general convergence
    proof using the asynchronous stochastic approximation framework.
    The treatment handles non-uniform visitation patterns and
    connects to the ODE method for stochastic approximation
    \citep{borkar2008}.

  \item \citet{jaakkola1994} gives an alternative convergence proof
    based on a direct contraction argument, which is more accessible
    but slightly less general.

  \item \citet[Chapter 6]{suttonbarto2018} provides extensive
    empirical comparisons of SARSA, Q-learning, and Expected SARSA
    on benchmark tasks including the cliff-walking and windy
    gridworld problems.

  \item For Deep Q-Networks (DQN), Double DQN, Dueling DQN, and
    related extensions, see Lessons in Phase 04 (Deep RL). The
    tabular convergence theory developed here provides the
    foundation for understanding why these deep RL methods work
    and when they can fail.

  \item \citet[Section 4]{szepesvari2010} provides a concise
    survey of finite-time convergence rates for TD control methods,
    complementing the almost-sure convergence results proved here.
\end{itemize}
